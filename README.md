# Tuber_Miner
An easy 'tube downloader

Windows setup:

1. Create a folder on your computer where you want this program to run

2. Right click inside new folder and click "Open Terminal" and enter:

git clone https://github.com/Andro-Meta/Tuber_Miner.git

2. Run setup.bat

3. Open the new "Dist" folder and run Tube_Miner.exe

# Using Tube_Miner
Copy the 'tube link you want to download, click Fetch Video Info
Choose if you want to download the full video or just a section.
Choose the file format you want to download.
Click download and wait. The file will show up next to the Tube_Miner.exe program
